# DeepFake Image Detection

This project detects whether an image is Real or Fake using a CNN model.

## Tech Stack
- Python
- TensorFlow / Keras
- FastAPI
- HTML, CSS, JavaScript

## Run Instructions
1. Train model: `python ml/train_model.py`
2. Start backend: `python -m uvicorn backend.app:app --reload`
3. Open frontend `index.html`
